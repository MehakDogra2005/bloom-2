Font Size Test Results

This page tests the standardized sidebar font sizes:

  • Logo: Uses --sidebar-logo-size (24px)
  • Navigation items: Uses --sidebar-nav-size (16px)
  • Dropdown items: Uses --sidebar-dropdown-size (14px)
  • Emoji icons: Uses --sidebar-emoji-size (16px)
  • Dropdown emojis: Uses --sidebar-dropdown-emoji-size (14px)

All font sizes are now consistently applied using CSS variables and respond to different screen sizes.
